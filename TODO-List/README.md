# Server TODO List

这个目录不是给用户手动查看或手动维护的，而是给助手在服务器端持续维护。

## 目标

- 记录 Zario 平时交代的待办事项
- 在用户询问“今天还有哪些没做的事情”时，直接根据这里回答
- 用户不会自己打开这些文件，也不会自己手动记录
- 记录、整理、更新状态都由助手完成

## 文件说明

- `tasks.md`：主任务清单，统一维护所有待办
- `today.md`：今天需要优先关注的事项
- `done.md`：已完成事项归档
- `notes.md`：补充说明、上下文、提醒

## 助手维护规则

1. 用户说要记住/待办/提醒/之后做/别忘了时，主动写入任务清单
2. 当用户问“今天还有哪些没做的事情”时，优先读取 `today.md` 和 `tasks.md`
3. 已完成事项从待办移到 `done.md`
4. 如果没有明确截止时间，默认先放进 `tasks.md`
5. 如果明显是今天要做的，加入 `today.md`
